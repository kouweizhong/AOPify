**AOPify** is a lightweight AOP framework that provides basic, fluent style AOP features. It also provides custom logging capability

<strong>List of supported methods</strong>
  - Until
  - Delay
  - While
  - WhenTrue
  - Log
  - Before
  - After
  - HowLong
  - Catch
  - CatchAndThrow
  - ProcessAsync
  - Run
  - Return
  - RegisterLogger
  - WithTransaction
